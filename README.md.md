# Superstore Queries - Version Control Example

This folder contains two versions of the same SQL file to demonstrate version control on GitHub.

Files:
- sales_by_category_region_initial.sql  - Initial version (simple aggregation)
- sales_by_category_region_modified.sql - Modified version (adds state, profit margin, avg order value)
- README.md - this file
- CHANGELOG.md - small changelog with commit-like notes
